Detailed Description

A system controlling a dynamic physics world, only made if a gameobject in the layer has a rigidbody component.

This system controls the physics world. In order to be more efficient, all physics updates are run on a separate thread using the updateWorld function. Since this does not take much of the threads time, it also has the functionality of the ObjectUpdateThread class in order to update objects in the same way when it doesn't need to update the physics.
